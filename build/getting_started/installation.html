Install CADET-Python¶

The easiest way to create CADET simulations is to use the CADET-Python interface. For this purpose, we recommend installing Anaconda. Anaconda is a high-performance scinetific installation of Python that includes many common packages needed for scientific and engineering work. Download the installer from their website and run it for the local user. To install the CADET-Python, open an Anaconda Shell and run:

conda install -c immudzen cadet

If you would also like to use CADET-Match for parameter estimation, also run:

conda install -c immudzen cadetmatch

To run our examples, we recommend also installing the following packages:

conda install jupyterlab scipy matplotlib gitpython
